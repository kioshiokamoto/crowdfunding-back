## Ejecutar localmente

Clonar proyecto

```bash
  git clone https://github.com/kioshiokamoto/strong-auth
```

Ir a ruta de proyecto

```bash
  cd strong-auth
```

Instalar dependencias

```bash
  npm install
```

<strong>Debe actualizar variables de entorno</strong>

Iniciar servidor

```bash
  npm run start
```
